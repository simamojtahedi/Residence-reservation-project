# Next.js Project
<br/>
این پروژه از پایگاه داده mongo.db استفاده میکند و دارای امکانات زیر است :
- صفحه اصلی
- صفحه اتاق‌ها
- صفحه اتاق تکی و توضیحات با امکان لایک و بوکمارک
- صفحه درباره ما
- صفحه تماس با ما
- صفحه ورود و عضویت
